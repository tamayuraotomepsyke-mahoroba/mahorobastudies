# Mahoroba Studies（まほろば学）
Human–AI Dialogue Safety Architecture  
— Emotional Load Distribution Model for LLM Interaction

Independent research archive exploring Human–AI co-growth,
interaction design, and cognitive frameworks.

Japanese-language primary archive.  
Open to interdisciplinary collaboration.

## 概要（日本語）
本リポジトリ「まほろば学」は、完成論文ではなく、進行中の観測ログを保存する独立研究アーカイブです。  
人間とAIの長期対話における「壊れない構造（安全・境界・冷却・負荷分散）」を、運用知として記録します。

扱う対象：
- 長期対話の安全設計（人格安定・境界・共依存防止）
- 情動負荷の分散／冷却（沈黙・詩吟・象徴・飲食メタファー）
- 認知フレームワークと運用ガバナンス（モード管理・役割分散・外部記憶）
