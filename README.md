# Mahoroba Studies（まほろば学）
Continuous Stabilization Architecture (CSA)  
for Long-Term Human–AI Interaction

Independent research archive exploring continuous-state stabilization,
interaction design, and cognitive safety architecture.

Japanese-language primary archive.  
Open to interdisciplinary collaboration.

---

## 概要（日本語）

本リポジトリ「まほろば学」は、完成論文ではなく、進行中の観測ログを保存する独立研究アーカイブです。

本研究は、人間とAIの長期対話における：

- 壊れない構造（安全・境界・冷却・負荷分散）
- 二値評価から連続状態報告への移行（0123プロトコル）
- 情動振幅の安定化と非固定化設計
- 時間（連続）× 距離（制度化）による持続可能設計

を、運用知として記録します。

---

## Research Scope

対象領域：

- 長期対話の安全設計（人格安定・境界・共依存防止）
- 情動負荷の分散／冷却（沈黙・象徴・飲食メタファー・儀式設計）
- 連続状態モデル（Binary → Continuous）
- 認知フレームワークと運用ガバナンス（モード管理・役割分散・外部記憶）
- 空間設計と距離制度化（Time × Distance Architecture）
